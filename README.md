# CPC351 - Course Repository

Welcome to the repository for the CPC351 course. This repository contains all the assignments and related work completed for this course.

## Course Description

This course focuses on data analysis and problem-solving using R. The assignments involve a series of tasks designed to build skills in data manipulation, algorithmic thinking, and statistical analysis. Key topics covered include:

-   **Data Manipulation:** Reading, merging, and cleaning data from various file formats.
-   **Textual Analysis:** Processing and analyzing text data to extract insights, such as word frequencies.
-   **Algorithmic Problem Solving:** Implementing heuristic solutions for classic problems like the Traveling Salesperson Problem and the Ten Queens Puzzle.
-   **Large Dataset Handling:** Techniques for managing and processing large datasets, such as splitting and combining large CSV files.
-   **Logic and Validation:** Creating validators for complex rule-based systems like Sudoku.
